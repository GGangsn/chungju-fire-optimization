# 🚒 AI 활용 충주시 고령자 맞춤형 소방 위험 군집 시뮬레이터
> 충청북도 충주시의 고령화 인구 데이터를 바탕으로 소방 위험 구역을 군집화하고, 동적 행정 경계(GeoJSON) 시각화를 제공하는 공간 분석 시뮬레이터입니다.

## 📌 프로젝트 소개
*   **분석 대상**: 충청북도 충주시 25개 읍면동 행정구역
*   **목적**: 고령자 소방 위험 측면에서 취약 구역을 K-Means 군집 분석을 통해 등급화하고, 실시간 모니터링이 가능한 동적 맵 시뮬레이션을 구현하여 소방 자원 배치의 의사결정을 지원합니다.
*   **핵심 철학**: 외부 패키지 의존성(`pandas`, `sklearn`, `folium` 등)을 완전히 배제하고, Python 표준 라이브러리만을 사용하여 분석 알고리즘과 HTML 맵 렌더러를 자체 제작(Zero-Dependency)했습니다.

---

## 🛠️ 주요 기능 및 기술 스택
1.  **K-Means 군집화 직접 구현**:
    *   65세 이상 인구 수와 고령인구 비율을 표준화(Standardization)하여 데이터 스케일링.
    *   3개 군집으로 분류 후, 군집별 평균 위험점수(MinMax 가중 평균)를 산출해 고/중/저위험군을 합리적으로 재판정.
2.  **행정구역 경계(GeoJSON) 융합**:
    *   충주시 행정 경계 폴리곤 데이터를 결합하여 구역별로 시각화.
3.  **실시간 인터랙티브 맵 (Leaflet CDN)**:
    *   초기 화면에서 25개 구역을 위험 등급별로 반투명하게(빨강/노랑/초록) 렌더링.
    *   **동적 클릭 피드백**: 지도 상의 임의 지점을 클릭하면 핀이 생성되며 반경 2.0km 범위가 그려집니다. 범위 내에 속하는 구역들은 **실시간으로 진하게 강조**되고, 범위 밖의 구역들은 **아주 연하게 페이드아웃** 처리되어 모니터링 효과를 극대화합니다.
    *   지도의 빈 곳을 더블클릭하거나 꽂힌 핀을 다시 클릭하면 초기 반투명 상태로 즉시 리셋됩니다.

## 📂 파일 구조
*   `충청북도 충주시_인구통계_20251231.csv`: 분석에 사용한 원본 데이터
*   `chungju_emd.geojson`: 충주시 25개 행정구역 폴리곤 데이터
*   `extract_geojson.py`: 공공 데이터에서 충주시 경계만 필터링하여 GeoJSON을 빌드하는 스크립트
*   `chungju_fire_cluster_simulator.py`: 군집화 분석을 실행하고 시각화 HTML을 자동 생성하는 메인 스크립트
*   `index.html`: 생성된 최종 시큐레이터 인터랙티브 맵 (GitHub Pages 호스팅용)

## 🚀 실행 및 시연 방법
1.  **시뮬레이터 빌드** (지도를 새로 생성하려는 경우):
    ```bash
    python chungju_fire_cluster_simulator.py --output index.html
    ```
2.  **시연**:
    *   `index.html` 파일을 웹 브라우저에서 열거나, GitHub Pages를 통해 배포된 웹 주소로 접속합니다.
    *   지도를 마우스로 클릭하여 핀을 꽂고 실시간 폴리곤 스타일 변경 인터랙션을 체험해 보세요.
